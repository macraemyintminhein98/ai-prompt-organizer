# PromptForge AI

PromptForge AI is a micro-SaaS web application designed to help AI enthusiasts, content creators, and developers quickly save, categorize, and search their AI prompts. Focus on speed and ease of use for managing your prompts for tools like ChatGPT and Midjourney.

**Features:**
- Save and organize prompts.
- Categorize and tag for easy retrieval.
- Fast, intuitive search functionality.
- Clean and minimalist user interface.
